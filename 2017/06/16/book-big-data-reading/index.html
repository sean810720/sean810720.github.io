「大數據」閱讀感想

2017/06/16
  • 叫你讀書不讀書

這本放在書櫃裡很久了，一直沒機會很仔細去翻它。

書的第一章就直接了當強調，在這個資訊比你所能想像還更多的年代，分析資訊間「關聯性」的意義遠比探討因果關係還來的重要，例如最經典的例子：「男人在大賣場買啤酒也會順便買尿布」，或者廉價航空票價的規律性波動等。總之可以簡單理解成：

大數據分析 = 在大量資料中找出規則，作為將來提出預測的參考

講白了就是 Data Mining。所以作者還順便戰了一下大數據分析 / 機器學習 / 人工智慧在普羅常識與實際上的定義，大致上為以下幾點：

一. 大數據分析是用來預測未來的，讓數字說話

二. 人工智慧的重點在教會電腦擬人化

三. 機器學習是人工智慧的一部分
